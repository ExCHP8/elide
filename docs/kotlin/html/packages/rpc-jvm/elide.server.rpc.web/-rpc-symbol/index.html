Enumerates byte symbols which are meaningful in the gRPC-Web Protocol; there are only two, DATA and TRAILER.

The TRAILER value is used to encode gRPC responses over regular HTTP/1.1-style responses, if needed. The DATA symbol is used to demarcate a data frame inside a gRPC Web request or response.
